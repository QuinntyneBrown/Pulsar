Pulsar — UI Design Mocks

Static, presentation-only reproductions of the Pulsar dashboard, reverse-engineered from the implemented Angular app under frontend/src/app. They share the real design system in frontend/src/styles.scss (consolidated here into mocks.css) so the look matches the running tool. No JavaScript or backend — these capture intended layout, states, and styling.
